Class Fetcher

Fetcher

The Fetcher object is a helper object for a quadstore which turns it from an offline store to an online store. The fetcher deals with loading data files rom the web, figuring how to parse them. It will also refresh, remove, the data and put back the data to the web.

load

  • load<T>(uri: T, options?: Partial<AutoInitOptions>): T extends (string | NamedNode)[] ? Promise<Response[]> : Promise<Response>
    • Defined in src/fetcher.ts:949

    Promise-based load function

    Loads a web resource or resources into the store.

    A resource may be given as NamedNode object, or as a plain URI. an array of resources will be given, in which they will be fetched in parallel. By default, the HTTP headers are recorded also, in the same store, in a separate graph. This allows code like editable() for example to test things about the resource.
